# CodeMirrorBundle

## Description

Code Mirror editor form field for symfony2.


## Installation.

Using composer

``` bash
$ composer require kamran/code-mirror-bundle dev-master
```
Add the CodeMirrorBundle to your AppKernel.php file:

```
new Kamran\CodeMirrorBundle\KamranCodeMirrorBundle();
```
